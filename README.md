# seminar-cpe-warunyu
 
Repository สำหรับเก็บงานวิชาการสัมมนาทางวิศวกรรมคอมพิวเตอร์
 
## เกี่ยวกับ Repository นี้
 
อธิบายสั้นๆ ว่า Repository นี้ใช้สำหรับเก็บซอร์สโค้ด ไฟล์โปรเจกต์ และประวัติการแก้ไข เพื่อให้สามารถพัฒนา ติดตามการเปลี่ยนแปลง และทำงานร่วมกับผู้อื่นได้

- `README.md` — เอกสารอธิบาย Repository
- `warunyu.md` — ไฟล์แนะนำตัว
